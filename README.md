# LinkTraces

A minimal personal link-in-bio page. Dynamically renders social/platform links from a constants file — add or remove a link by editing one line.

## Features

- Links rendered dynamically from `js/constant.js` — no hardcoded HTML buttons
- One-click copy to clipboard on each link
- Click profile image to copy the page URL
- Right-click and devtools keyboard shortcuts disabled

## Links configured

- YouTube
- Itch.io
- GitHub
- LinkedIn

## Stack

Vanilla HTML · CSS · JavaScript — no dependencies, no build step.
