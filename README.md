# LinkTraces

A free, self-hosted alternative to Linktree — built because Linktree's free tier offers almost no customization and I wanted full control over how my links looked and behaved.

Dynamically renders social/platform links from a constants file. Add or remove a link by editing one line. No account, no platform dependency, no locked features.

## Features

- Links rendered dynamically from `js/constant.js` — no hardcoded HTML buttons
- One-click copy to clipboard on each link
- Click profile image to copy the page URL
- Right-click and devtools keyboard shortcuts disabled

## Stack

Vanilla HTML · CSS · JavaScript — no dependencies, no build step, deploy anywhere.
